# FactorySimulator
A college project to simulate bearing factory proccess

- to build:
  ```
    cd src
    javac MainProgram.java
  ```
  
- to run:
  ```
    cd src
    java MainProgram [simulated time] [torno] [fresa] [mandril]
  ```
- example:
  ```
    java MainProgram 200 5 6 5
  ```
